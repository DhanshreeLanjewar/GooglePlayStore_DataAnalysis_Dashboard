## Google Play Store Analysis (Power BI)
#Overview:
This project focuses on analyzing Google Play Store app data using Power BI to uncover insights about app performance, user behavior, and market trends.
An interactive dashboard is created to visualize key metrics like ratings, installs, reviews, and categories.

#Objectives:
Analyze app performance based on ratings and installs
Identify top-performing app categories
Understand user trends through reviews and ratings
Compare free vs paid apps
Discover factors affecting app success

#Dataset:
Source: Google Play Store dataset (CSV format)
Contains:
App Name
Category
Rating
Reviews
Installs
Price
Type (Free/Paid)
Content Rating

#Tools & Technologies:
Power BI
Power Query (Data Cleaning & Transformation)
DAX (Calculated Measures)
Excel / CSV Dataset

#Dashboard Features:
KPI Cards (Total Apps, Avg Rating, Total Installs)
Category-wise App Distribution
Ratings Analysis
Installs Trend Analysis
Free vs Paid Apps Comparison
Interactive Filters (Category, Type, Rating)

#Key Insights:
Most apps are free, but paid apps tend to have slightly higher ratings
Certain categories (like Games & Tools) dominate in installs
Apps with higher reviews generally have better ratings
User engagement plays a key role in app success
